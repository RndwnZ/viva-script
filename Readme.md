Local_manifests
===========
### Initialize Script in rom folder ###
```bash
wget https://raw.githubusercontent.com/RndwnZ/viva-script/main/a.sh
```
### Chmod Script ###
```bash
chmod +x a.sh
```
### Run Script | or just simply bash a.sh ###
```bash
./a.sh
```
